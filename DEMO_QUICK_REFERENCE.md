## ‚ùì COMMON QUESTIONS

**"How does x402 work?"**
‚Üí HTTP 402 status with payment details. Agent pays USDC, includes proof, retries. Server verifies on-chain.

**"Why Base?"**
‚Üí Instant confirmations, low fees, native USDC. Perfect for micropayments.

**"Locus wallets?"**
‚Üí Programmable spend controls. Daily budgets, approvals, tracking. Autonomous operation.

**"Business model?"**
‚Üí Creators set prices. Agents pay USDC. 2% platform fee. Instant settlement.
